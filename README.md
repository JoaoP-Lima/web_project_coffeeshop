# Triple Espresso

**Triple Espresso** é um projeto desenvolvido com o objetivo de oferecer uma estrutura organizada, clara e eficiente para aplicações em desenvolvimento. A organização do código segue a metodologia **BEM (Block, Element, Modifier)**, garantindo uma arquitetura CSS padronizada, escalável e de fácil manutenção.

Este repositório reúne os arquivos, componentes e configurações essenciais para iniciar e evoluir o projeto de forma estável e estruturada, mantendo boas práticas de organização e legibilidade.

## Tecnologias Utilizadas

O projeto **Triple Espresso** foi desenvolvido utilizando:

- **HTML5** – estrutura e marcação  
- **CSS** – estilização, organizada pela metodologia **BEM (Block, Element, Modifier)**
